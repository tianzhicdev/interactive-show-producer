## 核心生成流程（强制顺序）

写任何选择时，必须先填这张表，再写 label 和 question：

1. 本场最高赌注是什么？（生死 > 使命 > 阵营 > 关系 > 脸面——问题必须挂在最高层级上，R1）
2. 从下方【原型菜单】选两个不同原型
3. 为每个选项写出：**卖点**（玩家选它的强烈理由）+ **代价**（选它失去什么）
4. question 压缩双边的卖点与代价（R7）
5. 最后才写 ≤8 字的 label（动词开头、具体动作、含对象/手段）

## 原型菜单（每个选项=可见卖点+可见代价；两选项取不同行）

| 原型 | 卖点 | 代价 | 文案骨架 |
|---|---|---|---|
| 正面强攻 | 立场/尊严/即时震慑 | 树敌、暴露 | 当面对质 |
| 以退为进 | 保身份/保资源/换时间 | 失面子、错失先机 | 退而保线 |
| 隐忍藏锋 | 情报、潜伏安全 | 屈辱、同流之嫌 | 虚与委蛇 |
| 先查后动 | 确定性、铁证 | 时间窗口流失 | 暗中查验 |
| 赌一把 | 大收益可能 | 失败即重创 | 夜闯库房 |
| 稳妥有价 | 保底结果 | 固定付出(钱/人情/把柄) | 花钱消灾 |
| 诱惑饮鸩 | 即时利益、捷径 | 道德/关系/长期隐患 | 收下密礼 |
| 牺牲舍身 | 保护他人/大局 | 自身重大损失 | 舍信救人 |
| 站队抉择 | 巩固一方 | 必然辜负另一方 | 站队沈氏 |
| 借刀巧计 | 不沾手 | 失控风险、欠人情 | 借刀递信 |
| 破釜沉舟 | 不可逆优势 | 断退路 | 焚信明志 |
| 表态本心 | 定义自我（低压节点专用） | 无机制代价 | 坦然相认 |

配对建议（高潮节点）：强攻×以退为进 / 牺牲×站队 / 赌×稳妥有价 / 诱惑×本心。
三选项节点：第三项必须测试**不同能力维度**，绝不允许置身事外项。

## 规则（R1-R14）

- **R1 最高赌注锚定**：问题挂在场景最高层级目标上；低层关切要入选项，必须先在文中升级其赌注。
  反例：手握精铁密信（使命级）却问「查铁证还是回怼立威」（脸面级）。
- **R2 无纯损选项**：每个选项必须**写明**它推进/保护什么。反例：「冷眼退避」（零收益的退）。
  正例：「退一步保身份」。
- **R3 收益表述强度对称**：两边卖点同等具体。一边"立刻赢"另一边"或许有用"=被支配选项
  （Dunyazad 实证：模糊收益被玩家当成坏选项）。
- **R4 确定性对称**：确定收益 vs 模糊收益会压垮两难（Papers, Please 倾斜机制）。一边不确定，
  另一边就给不确定因素或确定代价。
- **R5 人格投票测试**：爽感型/情感型/谨慎型三种玩家中，每个选项至少是一种人格的首选。
- **R6 禁止置身事外**：不准"走开/不理/旁观/沉默"——除非文案写明它买到了什么（Failbetter：
  逃逸选项稀释两难）。
- **R7 问题点明双边代价**：「为了A冒X的险，还是保B放弃Y？」压缩式。玩家必须在选前有参考依据。
- **R8 同级价值对撞**：真两难=同层级目标互撞（忠vs义、家人vs大局），不是使命vs面子。
- **R9 风险标价**：风险不同要写明，高风险必须明示更高回报。
- **R10 危险选项更性感**：冒险项写得更鲜活诱人（Jayanth："诱使玩家做坏决定"）；稳妥项体面
  而不窝囊。坏结局也要精彩。
- **R11 选错也推进**：没有"无事发生"或变相重选；选错=有代价地成功或跌入新麻烦（fail forward）。
- **R12 一场内回应**：下一场开头必须可见地回应所选（对手反应/局面变化）；承诺的长线后果必须兑现。
- **R13 合身份逻辑**：选项必须符合主角的使命、已知信息与人设——拿着叛国铁证的人不会把口角
  当优先级（玩家投诉："不合现实逻辑的答案破坏真实感"）。
- **R14 文案形式**：动词开头的具体动作（非态度词）；两 label 语法平行、长度差≤3字、≤8字、
  动作互斥；禁模板语。反例：「冷眼退避」（态度，无动作无对象）。

## 校验分工

确定性（D 检查/lint）：
- 裸置身事外词作完整 label（退避/离开/不理/旁观/无视/沉默 等，无对象无收益从句）→ 违规
- 问题含对比结构（还是/宁可…还是）且 ≥2 个不同赌注名词短语；labels 动词开头、长度差≤3
- 两分支下一场内容非空且有差异（R11/R12 的结构半）

LLM 评审（S 检查细则）：
- R1 层级审计：列出场景在场目标按层级排序；问题是否挂最高层？
- R5 人格投票：三人格各自首选；任一选项零票 → 违规
- R2/R3 收益审计：引用每个选项写明卖点/代价的原文；两边具体度是否对称？
- R4 确定性分类；R8 价值层级命名；R10 哪个选项更鲜活（必须是更险的或持平）
- R13 合理性：given 使命/已知信息/人设，每个选项是否可信？
